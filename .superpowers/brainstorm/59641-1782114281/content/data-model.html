p class="subtitle">SwiftData 模型设计

核心数据模型
📚 Book
id: UUID
title: String
author: String?
coverPath: String?
filePath: String
fileType: Enum (.epub/.mobi/.pdf)
lastRead: Date?
progress: Double (0-1)
isFavorite: Bool
addedAt: Date
🔖 Bookmark
id: UUID
book: Book (关系)
position: String (位置标识)
chapter: String?
note: String?
createdAt: Date
🖍 Highlight
id: UUID
book: Book (关系)
selectedText: String
color: Enum (yellow/green/orange/blue)
startOffset: Int
endOffset: Int
chapter: String?
note: String?
createdAt: Date
